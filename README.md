# Astronomy Dashboard
  
## URL
  
> [Astronomy Dashboard](https://astronomy-web-production.up.railway.app/)  
  
## Intro  
  
This application visualizes a dataset of 16 000+ Near-Earth Objects (NEOs) recorded by NASA, i.e. asteroids or comets whose orbits come close enough to Earth to pose a potential collision danger. Each NEO contains a set of attributes describing its physical properties as well as data describing its orbit around the Sun.  
  
The application also contains a table displaying the physical properties of all NEOs in the database, and an interactive timeline of historical 'close-approach' events whereby a Near-Earth Object came unsually close to Earth during its orbit.

### Features
  
- Table displaying the physical properties of all NEOs in the database
- Search bar with autocomplete feature to select any NEO
- Dynamic description of the selected NEO, generated by Google's Gemini LLM
- 3D projection of the selected NEO's orbit around the Sun
- Timeline of historical 'close-approach' events between years 1900-2026
  
---
  
### How to Use

The NEOs page, which can be accessed from the navigation bar, renders a table of all NEOs in the database along with their physical properties.  
  
To view the data on the Dashboard, users can select a Near-Earth Object in the search bar. It has an autocomplete feature so it's not necessary to know their full names.  
  
When a NEO is selected, the Orbit View will render an interactive 3D view of its orbit around the Sun. The view can be dragged, tilted, and zoomed in/out.  
  
If the NEO has one or more recorded close-approach events since the year 1900, the Timeline will automatically skip to the year of the event and highlight it.  
  
The timeline can also be interacted with on its own without first selecting a NEO, by browsing or entering a year in the search bar. Hovering over an event will render a hoverbox displaying event data.
